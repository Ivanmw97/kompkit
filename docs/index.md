# KompKit Core

Tiny, cross-platform utilities for **Web (TypeScript)** and **Android (Kotlin)**.

Status: `0.0.x-alpha` (APIs may change).

## Utilities

- `debounce` — Debounce a function call by a configurable delay.
- `isEmail` — Validate if a string matches a basic email pattern.
- `formatCurrency` — Format numbers as localized currency strings.
